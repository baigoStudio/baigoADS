SSO 设置

baigo ADS 的用户以及后台登录需要 baigo SSO 支持。

 
填写说明

API 接口 URL

baigo SSO API 接口的 URL

APP ID

baigo SSO 应用的 APP ID

APP KEY

baigo SSO 应用的 APP KEY

同步登录

如为开启状态，当用户在本站登录的时候，所有部署在 baigo SSO 下的网站将同步登录，当用户切换到这些网站时，无需再次登录。

点击“保存“，成功后点击“下一步“。

SSO 升级

baigo SSO 的升级与 baigo ADS 的升级并无直接关联，如果您要检查 baigo SSO 是否可升级，请点击 SSO 升级。
