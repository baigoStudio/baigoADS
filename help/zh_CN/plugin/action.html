动作钩子 Action

动作是 baigo ADS 运行到某些环节，或者在某些事件发生时，就会被执行的一种钩子。任何的插件都可以通过动作钩子来指示系统在遇到这些环节或者事件的时候，就执行指定的 PHP 函数。

 
top
 

全局动作

名称 返回 说明
action_sys_runtime - 系统开始运行时触发
action_sys_plugin_activated - 所有插件激活以后触发
 
top
 

前台动作

名称 返回 说明
action_pub_init - 前台初始化时触发
 
top
 

管理后台动作

名称 返回 说明
action_console_init - 管理后台初始化时触发
action_console_advert_status 混合数组 修改广告状态时触发
action_console_advert_del 广告 ID 数组 删除广告时触发
 
top
 

后台界面动作

名称 返回 说明
action_console_head_before - 后台界面头部之前
action_console_head_after - 后台界面头部之后
action_console_navbar_before - 后台界面导航条之前
action_console_navbar_after - 后台界面导航条之后
action_console_menu_before - 后台界面菜单之前
action_console_menu_end - 后台界面菜单最后（用于增加一个菜单项）
action_console_menu_after - 后台界面菜单之后
action_console_foot_before - 后台界面底部之前
action_console_foot_after - 后台界面底部之后
 
top
 

安装 / 升级程序动作

名称 返回 说明
action_setup_dbtable_complete - 安装程序，生成数据表后触发。
action_upgrade_dbtable_complete - 升级程序，升级数据表后触发。
