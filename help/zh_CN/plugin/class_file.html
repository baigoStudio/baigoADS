文件系统类

系统内置了一个操作文件和目录的文件类，在需要时，随时可以实例化该类，该类名称为 CLASS_FILE，例如：$obj_file = new CLASS_FILE();。使用该类操作文件，推荐使用 绝对路径。

该类文件位于 ./core/class/file.class.php，在开发者实例化时，系统会自动加载该文件，无需再 include，更为详细的使用可以参考该文件的源代码。

 
top
 

属性

 
属性 类型 说明
status bool 操作状态 true 成功，false 失败。
perms int 新建目录的权限，默认为 0771。关于目录的权限请查看 PHP 官方网站
 
top
 

方法

 
方法 说明
dir_mk 创建一个目录
dir_list 列出一个目录
dir_del 删除一个目录
file_put 写入一个文件
file_read 读取一个文件
file_copy 复制一个文件
file_del 删除一个文件
 

dir_mk($path)

说明

创建一个目录。

参数

名称 类型 必需 描述
path string true 要创建的路径。如果目录不存在，则会逐层创建。

返回

true 成功，false 失败。

 
top
 

dir_list($path[, $ext = ''])

说明

列出一个目录

参数

名称 类型 必需 描述
path string true 要读区的路径。如果目录不存在，则会逐层创建。
ext string false 指定扩展名，只列出指定扩展名的文件。
 

返回结果

目录列表。

 
返回结果示例

Array (
    [0] => Array (
        [name] => default //文件名
        [type] => dir //类型 dir 为目录, file 为文件
    )

    [1] => Array (
        [name] => test.txt
        [type] => file
    )

)

dir_del($path)

说明

删除一个目录

参数

名称 类型 必需 描述
path string true 要删除的路径。如果目录存在文件和子目录将会逐层全部删除。
 

返回结果

true 成功，false 失败。

file_put($path, $content)

说明

写入一个文件

参数

名称 类型 必需 描述
path string true 要写入的路径。如果目录不存在，则会逐层创建。
content string true 要写入的内容。
 

返回结果

写入字节数。

文件内容

file_copy($path_src, $path_dst)

说明

复制一个文件

参数

名称 类型 必需 描述
path_src string true 源文件路经。
path_dst string true 目的文件路经。如果目录不存在，则会逐层创建。
 

返回结果

true 成功，false 失败。
